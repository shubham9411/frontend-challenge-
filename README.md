# Frontend Challenge

## Directory Structure

--- dist (Concatentated and minified files)

--- index.html

--- src (Source files)

--- [build tool files]

## Quick Start

Install dependencies:

```bash
$ npm install
```

Build and Start the server:

```bash
$ grunt
```
